## Beginner Exercises

### Exercise 1: Build a Blog

**Goal:** Create a simple blog with posts and comments

**Requirements:**
- List all posts
- Create new post
- Edit existing post
- Show post details
- List comments for each post

**Data:** Use https://api.fake-rest.refine.dev

**Time:** 1-2 hours

---

### Exercise 2: Add Authentication

**Goal:** Protect your blog with auth

**Requirements:**
- Login page
- Logout button
- Protect create/edit pages
- Show current user name

**Time:** 1 hour

---

## Intermediate Exercises

### Exercise 3: Add Filtering

**Goal:** Filter posts by category and author

**Requirements:**
- Category dropdown filter
- Author search filter
- Filter persistence in URL
- Clear filters button

**Time:** 1-2 hours

---

### Exercise 4: Build a Dashboard

**Goal:** Create analytics dashboard

**Requirements:**
- Total posts count
- Posts per category chart
- Recent posts list
- Top authors list

**Time:** 2-3 hours

---

## Advanced Exercises

### Exercise 5: Real-Time Comments

**Goal:** Add real-time updates to comments

**Requirements:**
- Setup Supabase
- Enable live mode for comments
- New comments appear automatically
- No page refresh needed

**Time:** 2-3 hours

---

### Exercise 6: Multi-Language Blog

**Goal:** Add i18n support

**Requirements:**
- English and Spanish
- Language switcher
- Translate all UI text
- Translate API responses

**Time:** 2-3 hours
